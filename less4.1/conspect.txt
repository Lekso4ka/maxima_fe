Отличия ЯП:
    компилируемые - человек пишет команду "сделай мне то-то и верни что-то". Эту команду берет компилятор (переводчик) и преобразовывает данные в код, понятный компьютеру (0001010101010), компьютер понимает что от него хотят, выполняет действие, отдает результат компилятору и тот выдает его человеку
        Objective-C (C)
        C# (.net) - прграммы MS Windows + игры
        Java (Kotlin)
        C++
        Pascal/Delphi
    интерпретируемые - есть программа, написанная на каком-то ЯП (смотри выше), внутри которой ряботает определенный набор команд - это интерпретатор
        Python
        JavaScript (node.js, браузер ПК)
        PHP (apache)
        go (golang)

Чтобы открыть консоль в браузере мы используем кнопку F12 или комбинацию Ctrl+Shift+i или открыть панель разработчика и найти вкладку console/консоль
